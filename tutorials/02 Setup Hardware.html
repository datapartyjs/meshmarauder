To get set up, you're going to need LoRA node hardware. Anything on the list of the Meshcore flasher will work, as lorapipe is based on a very stripped down copy of Meshcore.

Git clone lorapipe, and install PlatformIO, for example, on Ubuntu: 

    $ sudo apt install git platformio
    $ git clone https://github.com/datapartyjs/lorapipe/

Build and flash the firmware using PlatformIO: 

    $ cd lorapipe
    $ platformio run --environment Xiao_S3_WIO_Repeater
    $ platformio run --target upload --environment Xiao_S3_WIO_Repeater

Visual Studio Code's PlatformIO extension is extremely convenient, and will handle much of this for you!

Once you have your hardware flashed with lorapipe, you're ready to start recording and sending LoRA traffic. lorapipe gives users an interface to send and recieve raw bytes encoded using almost any LoRA modem preset (frequency, bandwidth, spread factor, coding rate, and sync word). This gives users the ability to send and recieve raw data over LoRA in a manner that is entirely agnostic of any overlying packet format.
