Once you have lorapipe flashed on your hardware, it's time to start capturing traffic. You will need to fetch MeshMarauder, find the serial port that your lorapipe node is running on, and then run the lora-scanner tool in the bin/ directory of the repository. You can run the following commands to do so: 

    $ git clone https://github.com/datapartyjs/meshmarauder
    $ cd meshmarauder
    $ dmesg    # run this after plugging in your lorapipe node, and the
               # /dev/tty device that appears is what you'll use
    $ bin/lora-scanner -D <tty> -p 917.25,500.0,7,8,2b

This will dump raw packets heard on the Defcon Meshtastic modem preset. You can record this to a file, or pipe it into something else.
