You can record Defcon MQTT traffic by running bin/mqtt-record You'll need to have mqtt-recorder installed for this script to work, which can be installed via PyPI. This will be left as an exercise for the reader. Once the required dependencies are met the output of this script can be saved to a file, or piped into something else.

One of the things that the output of this script can be piped into is the bin/mqtt-convert. This script is intended to convert the MQTT packets into the same raw format emitted by lorapipe. As with many of the other tools in this repository, mqtt-convert can be saved to a file, or piped into something else!
